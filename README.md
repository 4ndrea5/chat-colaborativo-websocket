Chat Colaborativo WebSocket

Proyecto de chat colaborativo en tiempo real utilizando WebSockets.

- Características
Comunicación en tiempo real entre múltiples usuarios
Asignación automática de nombre
Historial de mensajes
Notificación de conexión/desconexión

- Instalación
 1.-Instalar dependencias:
  npm install
2.-Ejecutar servidor WebSocket:
  node server/server.js
3.-Abrir en navegador:
  client/index.html
